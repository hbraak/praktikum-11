## So bearbeitet ihr eine Seite (ohne Git-Kenntnisse)

1. GitHub-Konto anlegen (kostenlos) und Heiko den Benutzernamen nennen → er trägt euch als Mitarbeiter ein
2. Seite anklicken → oben rechts auf das **Stift-Symbol** ✏️
3. Text ändern/ergänzen (Markdown: `**fett**`, `- Liste`, Tabellen wie oben)
4. Unten grüner Knopf **„Commit changes"** — kurze Beschreibung eintragen, fertig

Jede Änderung wird mit Name und Datum protokolliert, nichts geht verloren (Historie: Knopf „History" oben rechts).

## Konventionen

- **Jahrgangsnotizen** immer mit Datum + Kürzel: `[2026-07, BRAK] Gruppe 3 brauchte …`
- Erledigte Änderungswünsche in die Anleitung übernehmen (→ .tex im Repo) und hier abhaken
- Fotos von Aufbauten gerne per Drag & Drop in die Seite ziehen
